Visão Geral

  <groupId>io.github.sidneyroberto9</groupId>
  <artifactId>utils</artifactId>
  <version>1.0.4</version>
</dependency>

Spring Boot — Auto-configuração

Todos os services são registrados automaticamente como beans via auto-configuração. Nenhuma anotação necessária no projeto.

// Injeção direta no seu service
@Service
public class OrderService {
    private final CPFService cpfService;

    public OrderService(CPFService cpfService) {
        this.cpfService = cpfService;
    }
}
CPF

CPFService

br.io.sidneyroberto9.utils.cpf

Formatação e validação de CPF. Aceita entrada com ou sem máscara.

Método Retorno Descrição
format(String cpf) String Formata para XXX.XXX.XXX-XX. Aceita dígitos ou mascarado.
strip(String cpf) String Remove a máscara e retorna apenas os 11 dígitos.
isFormatted(String cpf) boolean Verifica se está no padrão XXX.XXX.XXX-XX.
isValid(String cpf) boolean Valida CPF pelo algoritmo de dígitos verificadores. Rejeita sequências repetidas.

Exemplo

CPFService cpf = new CPFService();

cpf.format("12345678909");      // "123.456.789-09"
cpf.strip("123.456.789-09");    // "12345678909"
cpf.isValid("12345678909");     // true
cpf.isValid("11111111111");     // false — sequência repetida
cpf.isFormatted("123.456.789-09"); // true

PhoneNumberService

br.io.sidneyroberto9.utils.phoneNumber

Formatação e validação de números de telefone brasileiros. Detecta automaticamente celular (9 dígitos) e fixo (8 dígitos). Suporta entrada com DDI +55.

Método Retorno Descrição
format(String phone) String Detecta celular (XX) XXXXX-XXXX e fixo (XX) XXXX-XXXX. Trata prefixo +55.
formatWithCountryCode(String phone) String Formata com DDI: +55 (XX) XXXXX-XXXX.
strip(String phone) String Remove tudo exceto dígitos.
getDDD(String phone) String Extrai os 2 dígitos do DDD. Retorna null se o número tiver menos de 10 dígitos.
isMobile(String phone) boolean Verifica se é celular — 9 dígitos locais iniciando com 9.
isLandline(String phone) boolean Verifica se é fixo — 8 dígitos locais.
isFormatted(String phone) boolean Verifica se já está no formato com DDD e hífen.
isValid(String phone) boolean Valida número e DDD contra lista oficial de DDDs brasileiros.

Exemplo

PhoneNumberService phone = new PhoneNumberService();

phone.format("83986635812");              // "(83) 98663-5812"
phone.format("+5583986635812");           // "(83) 98663-5812"
phone.formatWithCountryCode("83986635812"); // "+55 (83) 98663-5812"
phone.getDDD("83986635812");              // "83"
phone.isMobile("83986635812");            // true
phone.isLandline("8332221234");           // true
phone.isValid("83986635812");             // true

CepService

br.io.sidneyroberto9.utils.cep.domain

Busca de endereço por CEP com estratégia de fallback entre 5 provedores externos. Se um provedor falhar ou não retornar dados, o próximo é tentado automaticamente.

ViaCEP OpenCEP BrasilCEP CEP.Rest Zippopotam
Método Retorno Descrição
lookup(String cep) Address Busca o endereço. Se não encontrado em nenhum provider, retorna Address com campos nulos (exceto CEP).

Address — Campos retornados

uf cep district municipality street complement ibge gia ddd siafi unit state region

Exemplo

CepService cep = new CepService();
Address addr = cep.lookup("58038-000");

addr.getStreet();       // "Avenida Epitácio Pessoa"
addr.getMunicipality(); // "João Pessoa"
addr.getUf();           // "PB"

DateService

br.io.sidneyroberto9.utils.date

Cálculo de dias úteis com calendário nacional brasileiro via jollyday. Suporta LocalDate, LocalDateTime e Date.

Método Retorno Descrição
isHoliday(LocalDate date) boolean Verifica se a data é feriado nacional. Sobrecarregado para LocalDateTime e Date.
isBusinessDay(LocalDate date) boolean Dia útil = não é feriado e não é fim de semana.
addBusinessDays(LocalDate, int days) LocalDate Adiciona n dias úteis. Negativo subtrai. Pula finais de semana e feriados.
subtractBusinessDays(LocalDate, int days) LocalDate Subtrai n dias úteis da data.
adjustToNextBusinessDay(LocalDate) LocalDate Avança para o próximo dia útil se a data não for dia útil.
adjustToPreviousBusinessDay(LocalDate) LocalDate Recua para o dia útil anterior se a data não for dia útil.
getFirstBusinessDayOfMonth(int year, int month) LocalDate Primeiro dia útil do mês.
getLastBusinessDayOfMonth(int year, int month) LocalDate Último dia útil do mês.
countBusinessDays(LocalDate from, LocalDate to) long Conta dias úteis de from (inclusive) a to (exclusivo).

Exemplo

DateService date = new DateService();
LocalDate hoje = LocalDate.now();

date.isBusinessDay(hoje);               // true/false
date.isHoliday(LocalDate.of(2025, 12, 25)); // true — Natal
date.addBusinessDays(hoje, 5);          // 5 dias úteis à frente
date.countBusinessDays(hoje, hoje.plusDays(30)); // ~21
date.getFirstBusinessDayOfMonth(2025, 1); // 2025-01-02

StringUtils

br.io.sidneyroberto9.utils.strings

Manipulação de strings com suporte a português: capitalização com conectivos, slugification, remoção de acentos e operações com e-mail.

Método Retorno Descrição
isBlank(String value) boolean Null-safe. true se nulo, vazio ou só espaços.
isNotBlank(String value) boolean Inverso de isBlank.
trimOrNull(String value) String Trim, retorna null se blank.
requireNonBlank(String, String message) String Lança IllegalArgumentException se blank.
capitalize(String word) String Primeira letra maiúscula, restante minúsculo.
capitalizeWords(String fullName) String Capitaliza cada palavra. Conectivos em PT-BR (de, da, do...) ficam em minúsculo.
firstTwoNames(String fullName) String Extrai e capitaliza os dois primeiros nomes.
digitsOnly(String input) String Remove tudo exceto dígitos 0-9.
alphanumericOnly(String input) String Remove tudo exceto letras e números (Unicode-aware).
removeAccents(String input) String Remove acentuação via normalização NFD.
slugify(String input) String Gera slug URL-friendly: lowercase, sem acento, espaços → hífens.
truncate(String value, int maxLength) String Trunca em maxLength chars sem quebrar palavras.
containsIgnoreCase(String text, String search) boolean Busca case-insensitive. Null-safe.
normalizeEmail(String email) String Trim + lowercase.
emailDomain(String email) String Extrai o domínio em uppercase: user@gmail.com → GMAIL.
maskEmail(String email) String Mascara o email: u***@gmail.com.
encodeUrl(String baseUrl, String text) String URL-encoda o texto e concatena à URL base.

Exemplo

StringUtils str = new StringUtils();

str.capitalizeWords("joão da silva");   // "João da Silva"
str.slugify("Olá Mundo!");             // "ola-mundo"
str.removeAccents("café");            // "cafe"
str.maskEmail("usuario@gmail.com");   // "u***@gmail.com"
str.firstTwoNames("João da Silva Souza"); // "João Da"
str.truncate("texto longo", 5);       // "texto"
