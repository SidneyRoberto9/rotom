# HashUtils.sha512(byte[] data)

**Pacote:** `io.github.sidneyroberto9.rotom.hash`
**Arquivo:** `HashUtils.java`

## Descrição

Atalho para [`hash(data, "SHA-512")`](hash.md).

## Parâmetros

| Nome | Tipo | Significado |
|---|---|---|
| `data` | `byte[]` | bytes a hashear |

## Retorno

`String` — hash SHA-512 em hexadecimal minúsculo (128 caracteres).

## Comportamento de borda

Era o algoritmo padrão usado internamente pelo `HashCheckSumUtil.generateHash(file)` original
(quando nenhum algoritmo era especificado) — aqui é sempre explícito, sem "algoritmo padrão"
implícito.

## Exemplo

```java
HashUtils hash = new HashUtils();

hash.sha512("hello world".getBytes());
```
