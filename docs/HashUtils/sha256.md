# HashUtils.sha256(byte[] data)

**Pacote:** `io.github.sidneyroberto9.rotom.hash`
**Arquivo:** `HashUtils.java`

## Descrição

Atalho para [`hash(data, "SHA-256")`](hash.md).

## Parâmetros

| Nome | Tipo | Significado |
|---|---|---|
| `data` | `byte[]` | bytes a hashear |

## Retorno

`String` — hash SHA-256 em hexadecimal minúsculo (64 caracteres).

## Comportamento de borda

Sem particularidades além das já descritas em [`hash`](hash.md).

## Exemplo

```java
HashUtils hash = new HashUtils();

hash.sha256("hello world".getBytes());
// "b94d27b9934d3e08a52e52d7da7dabfac484efe37a5380ee9088f7ace2efcde"
```
